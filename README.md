i2x-client
==========

**i2x** client agent. Sample Ruby to create **i2x** distributed agent running locally.

## Installation

Add the _i2x_ gem:

    gem install i2x

And then execute:

    $ ruby i2x.rb <config_file>

## Usage

### Scheduling with cron

To perform scheduled checks, enable running this client as a cron job.
